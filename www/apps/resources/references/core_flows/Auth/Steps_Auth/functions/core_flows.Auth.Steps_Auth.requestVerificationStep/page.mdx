# requestVerificationStep - Medusa Core Workflows Reference

This documentation provides a reference to the `requestVerificationStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step requests authentication verification.

:::note

This is available starting from [Medusa v2.15.5](https://github.com/medusajs/medusa/releases/tag/v2.15.5).

:::

```ts title="src/workflows/my-workflow.ts"
import { createWorkflow } from "@medusajs/framework/workflows-sdk"
import { requestVerificationStep } from "@medusajs/medusa/core-flows"

const myWorkflow = createWorkflow(
  "my-workflow",
  () => {
    const data = requestVerificationStep({
      "provider": "{value}",
      "entity_id": "{value}"
    })
  }
)
```
