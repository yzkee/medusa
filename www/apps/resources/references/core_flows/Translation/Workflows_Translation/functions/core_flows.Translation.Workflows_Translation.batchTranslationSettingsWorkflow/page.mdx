# batchTranslationSettingsWorkflow - Medusa Core Workflows Reference

:::note

To use this resource, make sure to [enable its feature flag: translation](https://docs.medusajs.com/development/feature-flags/toggle)

:::

This documentation provides a reference to the `batchTranslationSettingsWorkflow`. It belongs to the `@medusajs/medusa/core-flows` package.

This workflow creates, updates, and deletes translation settings in batch.
It's used by the [List Translation Settings API route](https://docs.medusajs.com/api/admin#translations\_gettranslationssettings).

You can use this workflow within your own customizations or custom workflows, allowing you
to create, update, and delete translation settings in your custom flows.

:::note

This is available starting from [Medusa v2.13.0](https://github.com/medusajs/medusa/releases/tag/v2.13.0).

:::

```ts title="src/api/workflow/route.ts"
import type {
  MedusaRequest,
  MedusaResponse,
} from "@medusajs/framework/http"
import { batchTranslationSettingsWorkflow } from "@medusajs/medusa/core-flows"

export async function POST(
  req: MedusaRequest,
  res: MedusaResponse
) {
  const { result } = await batchTranslationSettingsWorkflow(req.scope)
    .run({
      create: [{
        entity_type: "product",
        fields: ["title", "description"],
        is_active: true
      }],
      update: [{
        id: "ts_123",
        is_active: false
      }],
      delete: ["ts_456"]
    })

  res.send(result)
}
```
