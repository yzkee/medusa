# updateReturnReasonsWorkflow - Medusa Core Workflows Reference

This documentation provides a reference to the `updateReturnReasonsWorkflow`. It belongs to the `@medusajs/medusa/core-flows` package.

This workflow updates return reasons matching the specified filters. It's used by the
[Update Return Reason Admin API Route](https://docs.medusajs.com/api/admin#return-reasons\_postreturnreasonsid).

You can use this workflow within your customizations or your own custom workflows, allowing you to
update return reasons within your custom flows.

<SourceCodeLink link="https://github.com/medusajs/medusa/blob/699a9a85c795e13f9e787056e0fb65cfea0115dc/packages/core/core-flows/src/return-reason/workflows/update-return-reasons.ts#L57" />

  req: MedusaRequest,
