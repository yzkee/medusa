# batchPropertyLabelsWorkflow - Medusa Core Workflows Reference

:::note

To use this resource, make sure to [enable its feature flag: view_configurations](https://docs.medusajs.com/development/feature-flags/toggle)

:::

This documentation provides a reference to the `batchPropertyLabelsWorkflow`. It belongs to the `@medusajs/medusa/core-flows` package.

This workflow creates, updates, or deletes property labels in batch.

:::note

This is available starting from [Medusa v2.13.7](https://github.com/medusajs/medusa/releases/tag/v2.13.7).

:::
