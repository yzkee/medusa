# verifyAuthMfa - Auth Module Reference

This documentation provides a reference to the `verifyAuthMfa` method. This belongs to the Auth Module.

<Note>

You should only use this methods when implementing complex customizations. For common cases, check out [available workflows instead](/medusa-workflows-reference).

</Note>

This method verifies a pending multi-factor authentication (MFA) setup.

:::note

This is available starting from [Medusa v2.15.3](https://github.com/medusajs/medusa/releases/tag/v2.15.3).

:::

## Example

```ts
const mfaConfig = await authModuleService.verifyAuthMfa({
  id: "mfa_123",
  code: "123456"
})
```
