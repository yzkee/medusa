# AuthMfaProviderMethod

 **AuthMfaProviderMethod**: `"totp"` \| `string` & `object`

Supported multi-factor authentication (MFA) providers. Defaults to time-based one-time password (TOTP) but allows string extensions.

:::note

This is available starting from [Medusa v2.15.3](https://github.com/medusajs/medusa/releases/tag/v2.15.3).

:::
