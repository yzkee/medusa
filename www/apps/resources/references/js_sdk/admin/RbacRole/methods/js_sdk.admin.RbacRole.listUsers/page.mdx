# listUsers

This method retrieves a paginated list of users associated with an RBAC role.
It sends a request to the List Role Users API route.

## Example

```ts
sdk.admin.rbacRole.listUsers("role_123")
.then(({ users, count, limit, offset }) => {
  console.log(users)
})
```
