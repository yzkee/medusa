# create

This method creates an RBAC role. It sends a request to the Create RBAC Role
API route.

:::note

This is available starting from [Medusa v2.15.5](https://github.com/medusajs/medusa/releases/tag/v2.15.5).

:::

## Example

```ts
sdk.admin.rbacRole.create({
  name: "Editor",
  description: "Can edit content"
})
.then(({ role }) => {
  console.log(role)
})
```
