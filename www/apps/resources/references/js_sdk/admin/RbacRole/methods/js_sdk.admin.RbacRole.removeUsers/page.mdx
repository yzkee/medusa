# removeUsers

This method removes users from an RBAC role. It sends a request to the
Remove Role Users API route.

## Example

```ts
sdk.admin.rbacRole.removeUsers("role_123", {
  users: ["user_123"]
})
.then(({ deleted }) => {
  console.log(deleted)
})
```
