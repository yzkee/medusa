# addUsers

This method adds users to an RBAC role. It sends a request to the
Add Role Users API route.

## Example

```ts
sdk.admin.rbacRole.addUsers("role_123", {
  users: ["user_123"]
})
.then(({ users }) => {
  console.log(users)
})
```
