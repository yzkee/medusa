# removeRoles

This method removes roles from a user. It sends a request to the
Remove User Roles API route.

## Example

```ts
sdk.admin.user.removeRoles("user_123", {
  roles: ["role_123"]
})
.then(({ deleted }) => {
  console.log(deleted)
})
```
