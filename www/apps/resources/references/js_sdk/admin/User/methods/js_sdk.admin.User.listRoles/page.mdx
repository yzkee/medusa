# listRoles

This method retrieves a paginated list of roles assigned to a user. It sends a request to the
List User Roles API route.

## Example

```ts
sdk.admin.user.listRoles("user_123")
.then(({ roles, count, limit, offset }) => {
  console.log(roles)
})
```
