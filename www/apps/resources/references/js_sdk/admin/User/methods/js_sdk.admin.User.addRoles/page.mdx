# addRoles

This method assigns roles to a user. It sends a request to the
Assign User Roles API route.

## Example

```ts
sdk.admin.user.addRoles("user_123", {
  roles: ["role_123"]
})
.then(({ roles }) => {
  console.log(roles)
})
```
