# {metadata.title}

Add [Klaviyo](https://www.klaviyo.com/) to build email lists and send marketing campaigns to your customers.

Klaviyo lets you send automated emails for abandoned carts, order confirmations, and marketing campaigns. Bloom sets it up for you in minutes.

## Prerequisites

Before adding Klaviyo, you need a [Klaviyo account](https://www.klaviyo.com/) and API keys.

To get your Klaviyo API keys:

1. Log in to your Klaviyo account.
2. Click your profile icon in the bottom left, then choose "Settings."
3. Click "API Keys" in the left sidebar.
4. Click "Create Private API Key."
5. Give your key a name (like "Bloom Store") and select the required permissions.
6. Copy your Private API Key.

You'll also need your Public API Key (also called Site ID), which you can find on the same page. You'll need both keys when Bloom asks for them.

## Add Klaviyo to Your Store

To add Klaviyo using the <InProductAction product={"bloom"} type={"MEDUSA_AI_OPEN_PANE"} data={{ pane: "integrations" }}>Integrations</InProductAction> panel:

1. Open the Integrations tab by pressing <Kbd>{getOsShortcut()}</Kbd> + <Kbd>k</Kbd> and selecting "Integrations."
2. Find the Klaviyo card in the available integrations.
3. Click "Build" on the Klaviyo card.

Bloom will start setting up Klaviyo and ask you questions about which features you want to enable.

## Answer Bloom's Questions

After you click "Build," Bloom asks questions to configure Klaviyo:

1. **Which events do you want to track in Klaviyo?** - List the events you want (like "order placed, abandoned cart, and new customer signup").
2. **Do you want to add a newsletter signup form?** - Answer "yes" if you want to collect email subscribers on your store.
3. **Where should the signup form appear?** - Specify the location (like "in the footer and at checkout").

After you answer these questions, Bloom asks for your Klaviyo API keys.

## Enter Your Klaviyo Credentials

When Bloom asks for your API keys, paste the keys you copied from your Klaviyo dashboard:

1. Enter your Private API Key (starts with `pk_`).
2. Enter your Public API Key or Site ID.

Bloom stores these keys securely and uses them to connect your store to Klaviyo.

<Note title="Technical Information">

Bloom stores your Klaviyo credentials as environment variables in your store. They're never exposed in your code or logs. See [Environment Variables](../../../../developers/environment-variables/page.mdx) for more details.

</Note>

## Test Klaviyo

After Bloom finishes the setup, test Klaviyo in your preview store:

1. Open your store preview.
2. Place a test order or trigger an event you configured.
3. Check your Klaviyo dashboard to see if the event was tracked.
4. If you added a newsletter signup form, test subscribing with a test email.
5. Verify the subscriber appears in your Klaviyo lists.

If events appear in your Klaviyo dashboard and signups work correctly, Klaviyo is working.

<Note>

**Test Email Campaigns:** You can create a test campaign in Klaviyo to make sure emails send correctly. Just make sure to use test email addresses, not real customer emails.

</Note>

## Publish Klaviyo Integration

Once you've tested Klaviyo and it's working correctly, publish your store to make it live with production credentials.

To publish your Klaviyo integration:

1. Click the "Publish" button at the top right of your project.
2. If you haven't published before, Bloom asks for details about your store.
3. Your Klaviyo API keys work for both test and live environments, so you don't need to enter new credentials.
4. Click "Publish" to make your changes live.

Your Klaviyo integration is now active on your live store. You can track real customer events and send marketing campaigns.

<Note type="warning">

**Review Your Campaigns:** Before publishing, make sure any automated email campaigns in Klaviyo are reviewed and ready to send to real customers.

</Note>
